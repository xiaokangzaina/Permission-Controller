## 项目介绍

这是一个为 AstrBot 准备的权限管理插件。

它可以帮助你更细致地控制机器人在群聊和私聊中的使用权限。你可以决定哪些普通好友可以私聊机器人，哪些 QQ 可以在指定群调用机器人，哪些群可以整体放行，以及哪些用户需要在群聊里被禁止调用。

这个插件的定位是“使用权限管理”，不是“管理员授权工具”。

私聊白名单里的用户只会被允许私聊机器人，不会因此获得 AstrBot 管理员权限，也不会拥有管理员指令权限。

---

## 核心功能

✨ 支持普通好友私聊白名单  
✨ 支持按 QQ 和群号组合放行  
✨ 支持整个 QQ 群聊放行  
✨ 支持群聊用户黑名单  
✨ 支持 AstrBot 管理员绕过限制  
✨ 支持将群聊放行列表同步到 AstrBot 平台白名单  

---

## 权限逻辑

管理员绕过限制开启后，AstrBot 平台管理员会优先放行。

私聊白名单只影响好友私聊。写在这里的 QQ 可以私聊机器人，但不会变成管理员。

放行权限 QQ 列表用于精确控制某个 QQ 在某个群里的调用权限。适合只允许指定用户在指定群调用机器人的场景。群聊调用权限规则开启后，未命中该列表且所在群也不在群聊整体放行列表中的用户会被拦截。

放行权限 QQ 群聊列表用于整体放行某些群。只有群号在这里时，整个群才可以通过本插件的群聊权限检查；没有填写的群不会整体放行。

群聊禁止调用黑名单用于禁止某些 QQ 在任何群聊中调用机器人。这里填写的是用户 QQ，不是群号。

## 配置项说明

| 页面显示名 | 用途 |
| :--- | :--- |
| 管理员绕过限制 | 开启后，AstrBot 平台管理员不会被本插件拦截。 |
| 管理员绕过唤醒词 | 开启后，AstrBot 平台管理员无需唤醒词或 @ 机器人，也能用普通消息唤醒机器人。 |
| 私聊白名单用户 | 填写允许私聊机器人普通用户的 QQ。不会授予管理员权限。 |
| 启用群聊调用权限规则 | 控制群聊放行规则是否生效。 |
| 放行权限 QQ 列表 | 按用户 QQ 和群号的组合，精确放行某个用户在某个群调用。 |
| 放行权限 QQ 群聊列表 | 填写群号后，该群聊整体放行，并会同步到 AstrBot 平台白名单。 |
| 启用群聊禁止调用黑名单 | 控制群聊用户黑名单是否生效。 |
| 群聊禁止调用黑名单 | 填写用户 QQ。命中的用户在群聊中会被禁止调用机器人。 |

## 推荐使用方式

如果你只想让少数好友私聊机器人，请把他们的 QQ 填到“私聊白名单用户”。

如果你想让某个群整体可用，请把群号填到“放行权限 QQ 群聊列表”。插件会在启动时尝试把这些群号同步到 AstrBot 普通配置里的平台白名单 ID 列表，这样通常不需要再去平台白名单里重复填写。

如果你想精确到某个人只能在某个群里调用，请使用“放行权限 QQ 列表”。填写时使用用户 QQ 和群号组合的形式，例如用户 3207189240 在群 1012112971 中放行，就填写 3207189240-1012112971。

如果你想禁止某些人在任何群聊里调用机器人，请把他们的 QQ 填到“群聊禁止调用黑名单”，并确认“启用群聊禁止调用黑名单”已经打开。

---

## 关于群聊整体放行

“放行权限 QQ 群聊列表”用于解决 AstrBot 平台白名单可能先于插件拦截的问题。

如果平台白名单开启，而某个群号没有在平台白名单中，那么消息可能在进入插件之前就被 AstrBot 核心拦截。为减少重复配置，本插件会在启动时把“放行权限 QQ 群聊列表”中的群号同步到平台白名单 ID 列表。

因此，你可以优先在本插件中维护群聊放行列表，让权限配置更集中。

---

## 私聊白名单说明

私聊白名单只代表允许指定 QQ 在好友私聊中使用机器人。

它不会让用户成为 AstrBot 管理员，不会放开管理员指令，也不会影响 AstrBot 平台配置中的管理员列表。

真正的管理员权限仍然来自 AstrBot 全局配置里的管理员 ID。

## 常见问题

### 为什么某个群没有按预期限制？

通常是因为这个群没有出现在放行权限 QQ 列表里，也没有出现在放行权限 QQ 群聊列表里。请先确认群号是否写对，以及是否重载过插件。

### 为什么黑名单里填群号没效果？

群聊禁止调用黑名单填写的是用户 QQ，不是群号。它的意思是禁止某个用户在群聊里调用机器人。

### 为什么管理员还能使用？

请检查“管理员绕过限制”。如果它开启，AstrBot 平台管理员会自动放行。

### 群临时会话为什么不按私聊白名单放行？

群临时会话不等于普通好友私聊。为了避免误放行，插件不会把群临时会话当成私聊白名单处理。

### 修改配置后为什么没立刻生效？

请在 AstrBot 后台重载插件，或者重启 AstrBot。涉及平台白名单同步时，重启通常更稳妥。

## 使用建议

建议把权限分成三层管理。

第一层是 AstrBot 平台白名单，用于决定哪些会话能进入 AstrBot。

第二层是本插件，用于决定哪些 QQ 或群聊可以调用机器人。

第三层是其他插件自己的权限配置，例如媒体解析、生图限制等。它们通常有各自独立的白名单或限制逻辑。

这样排查问题时会更清晰，也不容易把平台白名单、插件白名单和管理员权限混在一起。
